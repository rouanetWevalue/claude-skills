# Structure de l'analyse marché emploi

## Données à collecter

### 1. Contexte du poste cible
- Intitulés de poste équivalents (variants de nommage)
- Secteurs qui recrutent ce profil
- Types de structures (startup, ETI, grand groupe, cabinet conseil, public)

### 2. Rémunération
| Segment | Fourchette basse | Fourchette haute | Source | Date |
|---|---|---|---|---|
| Junior (0–3 ans) | | | | |
| Confirmé (3–7 ans) | | | | |
| Senior (7+ ans) | | | | |

Inclure : fixe, variable, avantages fréquents (remote, RTT, intéressement).

### 3. Compétences techniques demandées
Classer par fréquence dans les offres :
- **Indispensables** (>70% des offres) : ...
- **Appréciées** (40–70%) : ...
- **Différenciantes** (<40%, mais valorisées) : ...
- **En déclin** (présentes < 15%, tendance baisse) : ...

### 4. Compétences comportementales (soft skills)
Top 5 mentionnées dans les offres du secteur.

### 5. Tendances du marché
- Volume d'offres (en hausse / stable / en baisse) vs N-1
- Technologies émergentes dans ce domaine
- Impact IA sur le poste (augmentation / transformation / substitution partielle)
- Régions / villes les plus actives

### 6. Profil type des candidats retenus
- Niveau de formation habituellement requis
- Années d'expérience médiane pour le poste
- Certifications fréquemment mentionnées

## Format de sortie

Rapport structuré en 6 sections ci-dessus.
Conclure par : **Top 3 des quick wins** (compétences/signaux à acquérir ou mettre en avant pour maximiser la pertinence du profil sur ce marché).

## Règles

- Citer explicitement chaque source + date
- Distinguer France entière vs région spécifique si pertinent
- Signaler si les données > 12 mois (fiabilité réduite)
- Ne pas confondre "compétences listées sur LinkedIn" et "compétences réellement évaluées en entretien"
